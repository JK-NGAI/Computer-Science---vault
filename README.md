This repository contains a collection of Python programs and small systems developed to practice core programming concepts such as:

  -Functions
  -Dictionaries and data structures
  -File handling
  -User input and validation
  -Basic financial calculations
  -Menu-driven applications

The projects demonstrate how Python can be used to build simple real-world systems through command-line applications.

Projects in this Repository:

1. Banking Module Program:
   A menu-driven banking simulation that allows users to:
   -Create bank accounts
   -Deposit money
   -Withdraw money
   -View account details
  - Calculate loan interest

This project demonstrates:
   -Function design
   -Dictionary data structures for storing account data
   -Input validation and financial calculations

2. Catering Company Invoice Management System:
A command-line invoice system for a catering company.
   -Features include:
   -Create invoices
   -Add items with quantity and price
   -Automatic VAT calculation
   -View and update invoices
   -Delete invoices
   -Save invoices as text files

   Concepts demonstrated:
-File handling
-Data storage
-Menu-based program structure
-Data validation
-Invoice calculations

Learning Goals:
 The goal of these projects is to:
   -Practice Python programming fundamentals
   -Understand structured program design
   -Build simple real-world applications
   -Improve problem-solving and coding skills
